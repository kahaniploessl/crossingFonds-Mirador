# crossingFonds-Mirador
Crossing Fond's Mirador prototype via local server.

1. Install node.js

2. Open Terminal and cd into desired directory

```sh
cd Documents/CrossingFonds
```

3. Clone repository

```sh
git clone
```
    

4. cd into repo directory

```sh
cd crossingFonds-Mirador
```

5. Install Libraries

```sh
npm i
```

6. Start Server

```sh
npm start
```
